# Security policy

## Supported versions

Only the latest minor release of `btflow` receives security fixes.

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |
| < 0.1   | :x:                |

## Reporting a vulnerability

Please **do not** open a public issue for security-related bugs. Instead,
report them privately via GitHub's
[Security Advisories](https://github.com/rayanchatt/bachelor-thesis-flow/security/advisories/new)
flow. A maintainer will respond within seven days.

When reporting, include:
- a clear description of the issue and its impact,
- minimal steps to reproduce,
- the affected version and your environment, and
- any suggested mitigation if you have one.
